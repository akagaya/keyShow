# KeyShow

Windows用のキー入力オーバーレイ表示ツール。Claude Opus 4.6製。

## 機能

- キー入力をデスクトップ上に常時表示（2段構成バー）
  - 上段: 修飾キーインジケーター（Ctrl / Shift / Alt / Win）
  - 下段: 一般キー履歴（右寄せ、フェードアウト付き）
- 修飾キー同時押し表示（例: `Ctrl+C`）
- ドラッグ移動 / 辺・隅ドラッグでズーム
- トレイメニューから背景色・文字色・透過度を変更
- ウィンドウはフォーカスを奪わない

## 使い方

```
run.bat
```

初回実行時に仮想環境の作成と依存パッケージのインストールを自動で行います。  
終了はタスクトレイアイコンの右クリック→「終了」。

## exe ビルド

```
build.bat
```

`dist/KeyShow.exe` が生成されます。

## 依存

- Python 3.10+
- pynput / pystray / Pillow
